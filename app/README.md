# 动屏智创 MVP

这是“动屏智创：小学体育课堂互动大屏协同创作智能体”的第一版 Web 原型。

## 启动前端

```bash
npm install
npm run dev
```

## 启动真实 AI 代理

```bash
cd ../server
npm install
cp .env.example .env
npm run dev
```

需要配置：

- `AI_BASE_URL`
- `AI_API_KEY`
- `AI_MODEL`

前端开发环境会通过 Vite 代理把 `/api/*` 转发到 `http://127.0.0.1:8787`。

## 构建

```bash
npm run build
```

## 当前能力

- 三栏工作台：AI 对话修改区、项目结构与编辑区、互动大屏预览区。
- 内置主案例：三年级篮球运球接力挑战课。
- 大屏模块：课堂流程、规则说明、倒计时、计分器、分组轮换、路线图、安全提醒、体育精神。
- 模拟 AI 局部修改：规则、队伍、时间、路线、安全提醒、主题。
- 真实 AI 接入：通过本地轻后端代理调用 OpenAI 兼容接口，并返回结构化 patch。
- Patch 校验：非法时间、空队伍、删除安全模块、越界路线会被拦截或提示风险。
- 版本快照与回退。
- 运行记录：记录真实模型请求、AI 修改、手动编辑、计分、倒计时、导出和回退。
- 导出：Project State JSON、教案 Markdown、应用证据包 Markdown、单文件 HTML 大屏。

## 演示建议

依次点击左侧内置演示指令：

1. 把规则改得更适合三年级学生
2. 增加四组计分，每组10人，队名用动物名称
3. 把接力挑战倒计时改成8分钟
4. 路线图增加右侧返回，避免碰撞
5. 加入安全提醒，强调掉球后先观察再捡球
6. 改成高对比投屏风格

演示重点不是“AI 一次生成全部”，而是 Project State 被多轮局部修改，右侧大屏实时更新，并且可以回退和导出。

## 真实 AI / Mock 双通道

- 默认优先使用真实 AI。
- 如果后端不可用、模型未配置、网络失败或返回结构不合法，前端会自动回退到 Mock。
- Mock 不会被删除，它是演示兜底和调试兜底。

## 设计边界

当前版本已接入本地轻后端代理，用于调用 OpenAI 兼容模型并返回结构化 patch。系统仍然保留 Mock 兜底，不做后端持久化、不登录、不采集学生个人信息。
